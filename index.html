Profile

The Long Read

Started on the usual road — HTML, CSS, then React, then enough SQL to stop being afraid of a schema. The full-stack fundamentals came first, on purpose, before any of the chain work.

Solidity came next, and with it a different way of thinking: code that can't be quietly patched at 2am, state that everyone can see, mistakes that are permanent. Blockchain fundamentals — consensus, gas, wallets, the shape of a transaction — followed from there.

Rust is the current chapter, still being written.

"A web app forgives you. A smart contract doesn't. Learning to write for the second kind changed how I write the first." — R. Kapoor
