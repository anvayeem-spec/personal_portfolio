# ANVAYEE — Where Design Meets Code

> "What if a developer portfolio felt like the cover of a magazine instead of another landing page?"

That's the question that started this project.

After browsing countless developer portfolios, I noticed they often followed the same formula—hero section, skills grid, project cards, contact form. They were functional, but they rarely felt memorable.

I wanted to build something that reflected not only my work, but also my appreciation for thoughtful design.

Fashion magazines have always fascinated me because they know how to tell a story. Every headline has a purpose. Every image guides your attention. Every page feels intentional. I wondered:

**What if a portfolio could do the same?**

So instead of building another conventional portfolio, I designed mine as an editorial magazine.

Each section is treated like a feature article.
Typography becomes part of the experience.
Whitespace tells the story.
The portfolio reads like an issue rather than a website.

Behind the editorial aesthetic is my journey as a developer—building with HTML, CSS, React, SQL, and exploring blockchain, Solidity, Rust, and AI one project at a time.

This is **Volume 01**.
As I grow, so will every new edition.
